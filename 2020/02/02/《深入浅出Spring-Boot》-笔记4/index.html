MongoDB

NoSql数据库 和Redis区别： Redis主要放置在内存中 MongoDB是持久化Nosql数据库
是一种最像关系型数据库的Nosql

默认端口 27017
最好设置账号密码 因为是持久化数据库 不设置账号密码只能本地访问

安装

MongoDB 官网下载 安装 点击Mongo.exe 启动或者使用命令行窗口

Studio 3t 图形化工具 官网下载
